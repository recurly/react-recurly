# UseCheckoutPricingInput

<Subtitle slot={() => 'To interact with useCheckoutPricing'} />

A plain object used to set initial values for [useCheckoutPricing][use-checkout-pricing] and to update those values if needed with [setCheckoutPricing][use-checkout-pricing-set-pricing].
